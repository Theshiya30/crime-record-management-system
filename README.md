# 🚔 Crime Record Management System

## 📌 Overview
The Crime Record Management System is a Java-based application designed to manage criminal records efficiently. It allows users to store, retrieve, and manage crime-related data in a structured and organized manner. The system improves tracking of case details and ensures better data handling compared to manual methods.

---

## 🚀 Features
- Add and manage criminal records  
- Track case details efficiently  
- Structured data storage  
- Client-server interaction using Java  
- Simple and user-friendly interface  

---

## 🛠️ Technologies Used
- Java (Backend)
- HTML (Frontend)
- JavaScript (Client-side functionality)

## ▶️ How to Run

1. Compile the server:
javac src/CrimeRecordServer.java

2. Run the server:
java src/CrimeRecordServer

3. Open the client:
Open client/CrimeRecordClient.html in a browser
